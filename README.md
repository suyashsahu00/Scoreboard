# Basketball Scoreboard

A simple, minimalistic digital scoreboard for tracking basketball scores between Home and Guest teams. Built with vanilla HTML, CSS, and JavaScript.

## Features

- **Live Score Tracking**: Separate displays for Home and Guest scores.
- **Point Controls**: Dedicated buttons to add +1, +2, or +3 points for each team.
- **Digital Font**: Uses a retro "Cursed Timer" font for a realistic scoreboard look.
- **Responsive Design**: Clean layout that centers the scoreboard on the screen.

## Technologies Used

- HTML5
- CSS3 (Flexbox, Custom Fonts)
- JavaScript (DOM Manipulation)

## How to Use

1. Clone the repository or download the files.
2. Open `index.html` in your web browser.
3. Click the buttons to increment the scores.
